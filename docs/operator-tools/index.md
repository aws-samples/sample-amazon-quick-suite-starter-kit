# Operator Tools

Post-deployment management tools for Amazon Quick Suite.

## Available Runbooks

### [User Management Runbook](user-management-runbook.md)

Manage [AWS IAM Identity Center](https://docs.aws.amazon.com/singlesignon/) users and groups, and assign them to Quick Suite roles.

Use this runbook for:

- Creating groups and users
- Batch user provisioning
- Assigning groups to Quick Suite roles
- Managing user group memberships

### [Monitoring Runbook](monitoring-runbook.md)

Monitor Quick Suite usage, users, and account status.

Use this runbook for:

- Viewing account summaries
- Listing users and their roles
- Checking group memberships
- Tracking active vs inactive users
